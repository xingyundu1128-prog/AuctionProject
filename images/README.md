# Images Folder

Please add 4 sample images here:

1. `watch.jpg` - A vintage watch image
2. `keyboard.jpg` - A gaming keyboard image
3. `bike.jpg` - A mountain bike image
4. `painting.jpg` - An art painting image

**Image requirements:**
- File size: < 500KB each
- Format: JPG or PNG
- Aspect ratio: Roughly 4:3 or 1:1 works best

**Where to find images:**
- Unsplash.com (free high-quality images)
- Pexels.com (free stock photos)
- Or use your own photos

**Note:** Make sure you have the right to use the images for your school project.
